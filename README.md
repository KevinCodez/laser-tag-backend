# Laserbacks

## Building && Running the project:
### Mac
    cd into the directory for the project
    Run the following commands

    ./build.sh
    ./run.sh

### Windows
    
    cd into the directory for the project
    Run the following commands

    ./build.cmd
    ./run.cmd

### Navigate to:
    
    http://localhost:8080

### ***NOTE***: UDP traffic will only be accepted during 5 min window.
